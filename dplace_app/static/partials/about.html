About D-PLACE

D-PLACE is a Database of Phylogenies of Languages for the study of Cultural Evolution.

Source Code

D-PLACE is open source software. Source code is available on GitHub under an MIT License.

D-PLACE uses Travis CI for continuous integration. Automated tests are run with each commit to the repository, build status is reported at travis-ci.org/NESCent/dplace.

Please report issues using the GitHub Issue Tracker.
